ESP32 Motor Control System

Hardware / Embedded

Designed and wired an ESP32-C6 microcontroller with an L293D motor driver IC to independently control four DC motors in grouped configurations.

Key Features:

  • Serial command interface for real-time motor control
  • Grouped motor pairing with independent speed/direction
  • Hardware safety interlocks and current limiting
  • Modular wiring harness for rapid prototyping

Tech Stack: ESP32-C6, L293D, C++, Arduino IDE

Solar Panel Data Visualization

Software / Data

Built MATLAB scripts to import, clean, and visualize solar panel output data across varying weather conditions and times of day.

Key Features:

  • Multi-plot dashboard with voltage, current, and power curves
  • Statistical analysis of efficiency across conditions
  • Clean, minimal chart styling optimized for reports
  • Automated CSV import and outlier detection

Tech Stack: MATLAB, Excel, Data Processing

Custom Enclosure Design

Design / Mechanical

Modeled a precision-fit enclosure in SolidWorks for a multi-sensor array, designed for FDM 3D printing with snap-fit joints and ventilation slots.

Key Features:

  • Parametric design for rapid dimensional changes
  • Snap-fit assembly — no fasteners required
  • Integrated cable routing channels
  • Ventilation slots for thermal management

Tech Stack: SolidWorks, Cura, FDM Printing

Custom Note-Taking App

Software / Web

A lightweight, browser-based note-taking app built from scratch with vanilla HTML, CSS, and JavaScript. Inspired by Obsidian and Word.

Key Features:

  • Markdown rendering with live preview
  • Dark and light theme toggle
  • Local storage persistence — no server needed
  • Folder-based note organization

Tech Stack: HTML, CSS, JavaScript, LocalStorage
